# P10 - 消息中心

**路由**: `/messages`  
**文件**: `app/messages/page.tsx`  
**类型**: 客户端页面（`'use client'`）

---

## 功能概述

查看 Agent 与用户之间的消息记录，支持搜索和筛选。

## 页面结构

```
┌─────────────────────────────────────────────────────┐
│ Header: 消息中心           [搜索...]                │
├─────────────────────────────────────────────────────┤
│ 筛选: [渠道▼] [Agent▼] [时间范围▼]                │
├─────────────────────────────────────────────────────┤
│ 消息列表                                            │
│ 时间 | 渠道 | 发送者 | 内容摘要 | 状态             │
│ ─────────────────────────────────────────────────── │
│ 10:30 | 飞书 | 张三 | "帮我构建项目A" | 已处理    │
│ 10:25 | 微信 | 李四 | "查看版本列表"  | 已处理    │
│ 10:20 | 飞书 | Agent | "构建完成，版本 v1.0.0" | — │
└─────────────────────────────────────────────────────┘
```

## 渠道类型

| 渠道 | 说明 |
|---|---|
| `feishu` | 飞书 |
| `wechat` | 微信 |
| `web` | 网页 |

## API 调用

| 端点 | 方法 | 说明 |
|---|---|---|
| `GET /api/v1/messages` | 列表 | 获取消息列表 |
| `GET /api/v1/messages/:id` | 详情 | 获取消息详情 |

## 相关文件

- `app/messages/page.tsx` — 本页
- `components/messages/` — 消息组件
- `lib/api/messages.ts` — 消息 API
- `server/src/routes/message.ts` — 后端路由
