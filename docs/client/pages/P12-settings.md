# P12 - 设置页

**路由**: `/settings`  
**文件**: `app/settings/page.tsx`  
**类型**: 客户端页面（`'use client'`）

---

## 功能概述

用户个人设置，包括个人信息、通知设置、界面偏好。

## 页面结构

```
┌─────────────────────────────────────────────────────┐
│ Header: 设置                                        │
├───────────────┬─────────────────────────────────────┤
│ 设置分类       │ 设置内容                           │
│ ├─ 个人资料   │ 头像 | 姓名 | 邮箱 | 密码          │
│ ├─ 通知设置   │ 邮件 | 飞书 | 站内信              │
│ ├─ 界面偏好   │ 主题 | 语言 | 时区                 │
│ └─ 安全设置   │ 2FA | 会话管理 | API Key          │
└───────────────┴─────────────────────────────────────┘
```

## API 调用

| 端点 | 方法 | 说明 |
|---|---|---|
| `GET /api/v1/users/me` | 获取 | 获取当前用户信息 |
| `PUT /api/v1/users/me` | 更新 | 更新个人信息 |
| `PUT /api/v1/users/me/password` | 更新 | 修改密码 |

## 相关文件

- `app/settings/page.tsx` — 本页
- `components/settings/` — 设置组件
- `lib/api/users.ts` — 用户 API
- `server/src/routes/user.ts` — 后端路由
