# P06 - 定时任务

**路由**: `/cron`  
**文件**: `app/cron/page.tsx`  
**类型**: 客户端页面（`'use client'`）

---

## 功能概述

管理定时任务（Cron Job）的创建、编辑、启用/禁用、查看执行历史。

## 页面结构

```
┌─────────────────────────────────────────────────────┐
│ Header: 定时任务              [+ 新建定时任务]       │
├─────────────────────────────────────────────────────┤
│ 统计: 活跃 | 暂停 | 执行中 | 失败                    │
├─────────────────────────────────────────────────────┤
│ 定时任务列表                                        │
│ ├─ 每日构建 | */5 * * * * | active | 下次: 10:05  │
│ ├─ 每周备份 | 0 2 * * 0 | paused | —              │
│ └─ 每月报告 | 0 0 1 * * | active | 下次: 04-01   │
├─────────────────────────────────────────────────────┤
│ 执行历史                                            │
│ ├─ 每日构建 #45 | 成功 | 2024-01-15 10:00         │
│ └─ 每日构建 #44 | 失败 | 2024-01-15 09:55         │
└─────────────────────────────────────────────────────┘
```

## Cron 表达式

支持标准 cron 表达式，格式：`分 时 日 月 周`

| 示例 | 说明 |
|---|---|
| `*/5 * * * *` | 每 5 分钟 |
| `0 */2 * * *` | 每 2 小时 |
| `0 9 * * 1-5` | 工作日 9 点 |
| `0 0 1 * *` | 每月 1 日 |

## API 调用

| 端点 | 方法 | 说明 |
|---|---|---|
| `GET /api/v1/cron-jobs` | 列表 | 获取定时任务列表 |
| `POST /api/v1/cron-jobs` | 创建 | 新建定时任务 |
| `PUT /api/v1/cron-jobs/:id` | 更新 | 更新任务配置 |
| `DELETE /api/v1/cron-jobs/:id` | 删除 | 删除定时任务 |
| `POST /api/v1/cron-jobs/:id/enable` | 启用 | 启用任务 |
| `POST /api/v1/cron-jobs/:id/disable` | 禁用 | 暂停任务 |
| `GET /api/v1/cron-jobs/:id/runs` | 历史 | 获取执行记录 |
