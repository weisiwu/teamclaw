# P03 - 任务管理

**路由**: `/tasks` + `/tasks/[id]`  
**文件**: `app/tasks/page.tsx` + `app/tasks/[id]/page.tsx`  
**类型**: 客户端页面（`'use client'`）

---

## 功能概述

展示自动化任务列表，支持创建、查看、取消任务。任务由 Agent 接收后执行。

## 页面结构

### 列表页 (`/tasks`)

```
┌─────────────────────────────────────────────────────┐
│ Header: 任务管理              [+ 新建任务]          │
├─────────────────────────────────────────────────────┤
│ 筛选栏: [状态▼] [Agent▼] [排序▼] [搜索框]         │
├─────────────────────────────────────────────────────┤
│ 任务列表                                            │
│ ├─ T-001 | 构建任务 | running | Agent-1 | 10:30   │
│ ├─ T-002 | 测试任务 | pending | Agent-2 | 10:25   │
│ └─ T-003 | 部署任务 | completed| Agent-1 | 10:20 │
└─────────────────────────────────────────────────────┘
```

### 详情页 (`/tasks/[id]`)

```
┌─────────────────────────────────────────────────────┐
```

## 组件结构

| 组件 | 来源 | 说明 |
|---|---|---|
| `TaskTable` | `components/tasks/task-table.tsx` | 任务列表表格 |
| `TaskStatusBadge` | `components/tasks/task-status-badge.tsx` | 状态标签 |
| `useQuery / useMutation` | `@tanstack/react-query` | 数据获取/操作 |

## 任务状态

| 状态 | 说明 | 可执行操作 |
|---|---|---|
| `pending` | 等待分配 | 取消 |
| `running` | 执行中 | 取消 |
| `completed` | 已完成 | 重试 |
| `failed` | 失败 | 重试 |
| `cancelled` | 已取消 | — |

## API 调用

| 端点 | 方法 | 说明 |
|---|---|---|
| `GET /api/v1/tasks` | 列表 | 支持 `status`, `agentId`, `page`, `pageSize` 参数 |
| `GET /api/v1/tasks/:id` | 详情 | 获取单个任务 |
| `POST /api/v1/tasks` | 创建 | 创建新任务 |
| `DELETE /api/v1/tasks/:id` | 取消 | 取消任务 |
| `POST /api/v1/tasks/:id/retry` | 重试 | 重试失败任务 |
