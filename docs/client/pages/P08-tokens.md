# P08 - Token 管理

**路由**: `/tokens`  
**文件**: `app/tokens/page.tsx`  
**类型**: 客户端页面（`'use client'`）

---

## 功能概述

管理 API Token，支持创建、禁用、删除 Token，查看使用统计。

## API 调用

| 端点 | 方法 | 说明 |
|---|---|---|
| `GET /api/v1/tokens` | 列表 | 获取 Token 列表 |
| `POST /api/v1/tokens` | 创建 | 创建新 Token |
| `DELETE /api/v1/tokens/:id` | 删除 | 删除 Token |
| `POST /api/v1/tokens/:id/disable` | 禁用 | 禁用 Token |
| `GET /api/v1/token-stats` | 统计 | 获取使用统计 |
