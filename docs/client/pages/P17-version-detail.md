# P17 - 版本详情

**路由**: `/versions/[id]`  
**文件**: `app/versions/[id]/page.tsx`  
**类型**: 客户端页面（`'use client'`）

---

## 功能概述

查看版本的详细信息，包括截图、变更日志、版本对比、构建记录等。

## 页面结构

```
┌─────────────────────────────────────────────────────┐
│ Header: v1.0.0                    [编辑] [构建] [回退]│
├─────────────────────────────────────────────────────┤
│ 基本信息                                            │
│ 项目 | 标签 | 状态 | 创建时间 | 发布人 | Commit     │
├──────────────────────┬────────────────────────────┤
│ 版本截图              │ 变更日志                    │
│ [截图1] [截图2]      │ + 50 行 | - 20 行 | M 5 文件│
│                      │ [查看完整 diff]            │
├──────────────────────┴────────────────────────────┤
│ 版本时间线                                          │
│ ● 2024-01-15 10:00 创建版本                      │
│ ● 2024-01-15 10:05 构建成功                       │
│ ● 2024-01-15 10:10 部署到 production             │
├────────────────────────────────────────────────────┤
│ 版本对比                                            │
│ [选择版本 ▼] → v1.0.0  [对比]                     │
└────────────────────────────────────────────────────┘
```

## 组件结构

| 组件 | 说明 |
|---|---|
| `VersionHeader` | 版本头部信息 |
| `VersionScreenshots` | 截图展示 |
| `VersionChangelog` | 变更日志 |
| `VersionTimeline` | 版本时间线 |
| `VersionDiff` | 版本对比 |

## API 调用

| 端点 | 方法 | 说明 |
|---|---|---|
| `GET /api/v1/versions/:id` | 获取 | 获取版本详情 |
| `GET /api/v1/versions/:id/screenshots` | 截图 | 获取截图列表 |
| `GET /api/v1/versions/:id/changelog` | 变更日志 | 获取变更内容 |
| `GET /api/v1/versions/:id/diff?compare=:id` | 对比 | 与其他版本对比 |
| `GET /api/v1/versions/:id/timeline` | 时间线 | 获取版本事件时间线 |
