# P11 - 监控面板

**路由**: `/monitor`  
**文件**: `app/monitor/page.tsx`  
**类型**: 客户端页面（`'use client'`）

---

## 功能概述

系统状态监控面板，显示 Agent 状态、任务队列、性能指标。

## 页面结构

```
┌─────────────────────────────────────────────────────┐
│ Header: 系统监控           [刷新] [自动刷新▼]       │
├─────────────────────────────────────────────────────┤
│ Agent 状态                                          │
│ Agent-1 ● busy  | Agent-2 ● idle | Agent-3 ● busy │
├─────────────────────────────────────────────────────┤
│ 系统指标                                            │
│ CPU: ████░░ 45% | 内存: ███░░░ 60% | 网络: ██░░░░│
├─────────────────────────────────────────────────────┤
│ 任务队列                                            │
│ pending: 3 | running: 2 | completed: 156 | failed: 2│
├─────────────────────────────────────────────────────┤
│ 最近事件                                            │
│ 10:30 Agent-1 开始执行任务 T-045                   │
│ 10:25 任务 T-044 执行失败                          │
└─────────────────────────────────────────────────────┘
```

## 实时更新

- 自动刷新间隔：可配置（5s / 10s / 30s / 1min）
- 使用 React Query 的 `refetchInterval` 实现轮询

## API 调用

| 端点 | 方法 | 说明 |
|---|---|---|
| `GET /api/v1/monitor/status` | 获取 | 系统状态 |
| `GET /api/v1/agents/status` | 获取 | Agent 状态列表 |
| `GET /api/v1/monitor/events` | 获取 | 最近事件 |
