| ------------- | -------------- |

## PriorityBadge

显示消息/任务优先级标签。

| 属性     | 类型                           | 说明   |
| -------- | ------------------------------ | ------ |
| priority | `"P0" \| "P1" \| "P2" \| "P3"` | 优先级 |

优先级颜色：

- **P0**: 红色，最紧急
- **P1**: 橙色，紧急
- **P2**: 蓝色，一般
- **P3**: 灰色，较低
