# Sidebar 组件

## 功能说明

侧边导航栏组件，提供应用主要功能模块的导航。

## 引入

```tsx
import { Sidebar } from '@/components/layout/Sidebar';
```

## 位置

`components/layout/Sidebar.tsx`

## 主要功能

- 收起/展开切换
- 功能模块导航菜单
- 当前路由高亮
- 图标 + 文字显示模式

## 菜单项（典型）

- 项目概览 (Dashboard)
- 版本管理 (Versions)
- 分支管理 (Branches)
- 成员管理 (Members)
- 设置 (Settings)

## 使用示例

```tsx
<Sidebar />
```
