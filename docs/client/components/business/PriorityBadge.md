# PriorityBadge

消息优先级徽章组件。

## 导入

```tsx
import { PriorityBadge } from "@/components/messages/PriorityBadge";
```

## 概述

显示消息的优先级标签（如「紧急」「重要」「普通」）。

> 具体 Props 和变体请参考组件源码 `components/messages/PriorityBadge.tsx`。

## 使用示例

```tsx
<PriorityBadge priority="urgent" />
<PriorityBadge priority="normal" />
```
