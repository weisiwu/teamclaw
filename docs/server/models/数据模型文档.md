## 1. 存储层基础设施

### 1.1 `_migrations` — 迁移记录表

记录所有已执行的数据库迁移脚本。

| 字段                | 类型          | 必填 | 默认值      | 说明                            |
| ------------------- | ------------- | ---- | ----------- | ------------------------------- |
| `id`                | `SERIAL`      | ✅   | 自增        | 主键                            |
| `name`              | `TEXT`        | ✅   | —           | 迁移脚本名称（唯一）            |
| `executed_at`       | `TIMESTAMPTZ` | ✅   | `NOW()`     | 执行时间                        |
| `checksum`          | `TEXT`        | ❌   | —           | 脚本校验和                      |
| `execution_time_ms` | `INTEGER`     | ❌   | —           | 执行耗时（毫秒）                |
| `status`            | `TEXT`        | ✅   | `'success'` | 执行状态：`success` \| `failed` |

**索引**：`PRIMARY KEY (id)`，`UNIQUE (name)`

## 2. 用户与权限

### 2.1 `users` — 用户表

用户信息持久化存储，替代原有内存 Map 方案。

> 迁移：`20260324_001_create_users.sql`（修复 BUG-05）

| 字段         | 类型          | 必填 | 默认值     | 说明                                                  |
| ------------ | ------------- | ---- | ---------- | ----------------------------------------------------- |
| `id`         | `TEXT`        | ✅   | —          | 主键 UUID                                             |
| `user_id`    | `TEXT`        | ✅   | —          | 外部用户 ID（微信/飞书）                              |
| `name`       | `TEXT`        | ✅   | —          | 用户显示名称                                          |
| `role`       | `TEXT`        | ✅   | `'member'` | 角色：`owner` \| `admin` \| `member` \| `employee` 等 |
| `weight`     | `INTEGER`     | ✅   | `0`        | 权重值（影响消息优先级）                              |
| `wechat_id`  | `TEXT`        | ❌   | —          | 微信 OpenID                                           |
| `feishu_id`  | `TEXT`        | ❌   | —          | 飞书 OpenID                                           |
| `remark`     | `TEXT`        | ❌   | —          | 备注信息                                              |
| `created_at` | `TIMESTAMPTZ` | ✅   | `NOW()`    | 创建时间                                              |
| `updated_at` | `TIMESTAMPTZ` | ✅   | `NOW()`    | 更新时间                                              |

**索引**：

- `PRIMARY KEY (id)`
- `idx_users_user_id` ON `user_id`
- `idx_users_role` ON `role`
- `idx_users_wechat_id` ON `wechat_id`
- `idx_users_feishu_id` ON `feishu_id`
- `idx_users_created_at` ON `created_at DESC`
- `UNIQUE idx_users_wechat_id_unique` ON `wechat_id`（WHERE `wechat_id IS NOT NULL`）
- `UNIQUE idx_users_feishu_id_unique` ON `feishu_id`（WHERE `feishu_id IS NOT NULL`）

**外键**：无（`project_id` 为业务关联字段，未建外键约束）
