## 1. db.ts — PostgreSQL 连接池

**文件路径**: `server/src/utils/db.ts`

基于 `pg` 库的全剧连接池（`pool`），最多 20 个连接，空闲 30 秒自动回收。

```typescript
import { pool } from './utils/db.ts';
const { rows } = await pool.query('SELECT * FROM versions WHERE id = $1', [id]);
```

通过 `onShutdown` 注册优雅关闭（`pool.end()`）。

---

## 2. redis.ts — Redis 连接管理

**文件路径**: `server/src/utils/redis.ts`

基于 `ioredis` 的全局客户端，支持自动重连。

```typescript
import { redis } from './utils/redis.ts';
await redis.set('key', 'value', 'EX', 3600);
```

通过 `onShutdown` 注册优雅关闭（`redis.disconnect()`）。

## 3. chromadb.ts — Chroma 向量数据库客户端

**文件路径**: `server/src/utils/chromadb.ts`

封装 Chroma 单例客户端，提供 `getOrCreateCollection()` 方法。

```typescript
import { createChromaClient, getOrCreateCollection } from './utils/chromadb.js';
const collection = await getOrCreateCollection(`project_${projectId}`);
```

## 4. jwt.ts — JWT 签发与验证

**文件路径**: `server/src/utils/jwt.ts`

封装 `jsonwebtoken` 库，提供 Access Token（7天）和 Refresh Token（30天）签发与验证。

| 函数                        | 说明               | 有效期 |
| --------------------------- | ------------------ | ------ |
| `signToken(payload)`        | 签发 Access Token  | 7 天   |
| `signRefreshToken(payload)` | 签发 Refresh Token | 30 天  |
| `verifyToken(token)`        | 验证并解析 Token   | —      |

## 5. response.ts — 统一 API 响应格式

**文件路径**: `server/src/utils/response.ts`

提供标准化响应构建函数和 `ApiError` 类。

```typescript
// 成功
res.json(success({ id: 'v1.0.0' }));

// 错误
res.status(400).json(error(400, '参数校验失败', 'BAD_REQUEST'));

// Service 中抛出
throw new ApiError(404, 'NOT_FOUND', '版本不存在');
```

## 6. config.ts — 配置文件加载与校验

**文件路径**: `server/src/utils/config.ts`

从 `config.json` 加载配置并校验必填字段。

```typescript
import { loadConfig, validateConfig } from './utils/config.js';
const config = loadConfig();
const { valid, errors } = validateConfig();
```

## 7. config-validator.ts — 环境变量启动校验器

**文件路径**: `server/src/utils/config-validator.ts`

在服务启动时校验环境变量，不通过则 `process.exit(1)`。

```bash
node server/dist/utils/config-validator.js
```

**校验项**：必填变量（`PORT`、`NODE_ENV`）、推荐变量、API Key 格式（`sk-` 前缀）、JWT Secret 默认值警告。

---

## 8. shutdown.ts — 优雅关闭管理

**文件路径**: `server/src/utils/shutdown.ts`

管理 LIFO 顺序的资源关闭，注册 `SIGTERM` / `SIGINT` 信号监听。

```typescript
import { onShutdown, registerShutdownHandlers } from './utils/shutdown.js';

onShutdown('PostgreSQL', async () => {
  await pool.end();
});
onShutdown('Redis', async () => {
  redis.disconnect();
});

registerShutdownHandlers(); // 仅在 index.ts 调用一次
```
