## 项目概述

poetry-app 是一个中阿对照诗词阅读应用，包含 101 首精选诗词（唐、宋、元、明、清）。

### 现有数据

- **诗词数量**：101 首
- **诗人数量**：40 位
- **朝代**：唐(58)、宋(32)、清(5)、元(1)、明(2)
- **类型**：全部为诗（诗/词/曲待扩展）
- **翻译**：中阿对照

## 数据库设计

### 细粒度表结构

```
┌─────────────┐      ┌─────────────┐      ┌─────────────┐
│  dynasties  │      │    poets    │      │  poems     │
│─────────────│      │─────────────│      │─────────────│
│ id (PK)     │◄────│ dynasty_id  │      │ id (PK)     │
│ name         │      │ id (PK)     │◄────│ poet_id (FK)│
│ name_arabic │      │ name        │      │ title       │
│ period       │      │ bio         │      │ content[]   │
│ description  │      └─────────────┘      │ content_ar[]│
└─────────────┘                             │ ...        │
                                            └──────┬──────┘
                                                   │
                                            ┌──────▼──────┐
                                            │  poem_tags  │
                                            │─────────────│
                                            │ poem_id (FK)│
                                            │ tag_id (FK) │
                                            └──────┬──────┘
                                                   │
                                            ┌──────▼──────┐
                                            │    tags     │
                                            │─────────────│
                                            │ id (PK)     │
                                            │ name        │
                                            └─────────────┘

┌─────────────────┐
│  poetry_types   │
│─────────────────│
│ id (PK)         │
│ name (诗/词/曲) │
│ name_arabic      │
│ description      │
└─────────────────┘
         ▲
         │
   ┌─────┴─────┐
   │   poems    │
   │─────────────│
   │ type_id(FK)│
   └─────────────┘
```

## 技术栈

- **前端**: Next.js + React + Tailwind CSS
- **后端**: Next.js API Routes + Supabase
- **数据库**: PostgreSQL (Supabase)
- **部署**: Vercel
