# 08【P0】搜索功能开发

## 任务目标

开发诗词搜索功能，支持标题、诗人名、内容的实时搜索。

## 详细说明

### 8.1 搜索页布局

```
┌──────────────────────────────────────────────┐
│ [返回] ┌────────────────────────┐ [取消]     │
│        │ 🔍 搜索诗词或诗人...   │            │
│        └────────────────────────┘            │
├──────────────────────────────────────────────┤
│ 热门搜索                                    │
│ [李白] [杜甫] [静夜思] [春晓]              │
├──────────────────────────────────────────────┤
│ 搜索结果                                    │
│ ┌────────────────────────────────────────┐   │
│ │ 静夜思 - 李白                          │   │
│ └────────────────────────────────────────┘   │
└──────────────────────────────────────────────┘
```

### 8.2 搜索逻辑

- 实时搜索（debounce 300ms）
- 搜索标题、诗人名、内容
- 显示搜索结果数量
- 无结果时显示空状态

### 8.3 热门搜索

显示搜索频率最高的关键词，点击可直接搜索。

## 验收标准

- [ ] 搜索框正常输入
- [ ] 实时搜索正常
- [ ] 搜索结果正确
- [ ] 无结果空状态显示
- [ ] 热门搜索显示

## 技术栈

- Next.js
- React Query

任务 04（诗词 CRUD API）
