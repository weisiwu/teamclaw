# 05【P0】项目重构与目录结构调整

## 任务目标

将 poetry-app 项目重构为 Next.js + Tailwind CSS 技术栈，优化目录结构。

## 详细说明

### 5.1 技术栈调整

用户决定不使用 React Native + Expo，改为 **Next.js + Tailwind CSS**：
- 更好的 Web 端体验
- 更便捷的部署（Vercel）
- 与 teamclaw 项目技术栈统一

### 5.2 目录结构调整

```
poetry-app/
├── app/                    # Next.js App Router
│   ├── api/               # API 路由
│   │   ├── poems/
│   │   ├── poets/
│   │   └── dynasties/
│   ├── page.tsx           # 首页（诗词列表）
│   ├── poem/[id]/page.tsx  # 诗词详情
│   ├── search/page.tsx    # 搜索页
│   ├── layout.tsx
│   └── globals.css
├── components/            # React 组件
│   ├── ui/               # UI 基础组件
│   ├── poem/             # 诗词相关组件
│   └── layout/           # 布局组件
├── lib/                  # 工具库
│   ├── supabase.ts       # Supabase 客户端
│   └── api.ts            # API 调用封装
├── types/                # TypeScript 类型
│   └── index.ts
├── public/               # 静态资源
└── package.json
```

### 5.3 迁移现有代码

将现有代码迁移到新结构：
- `app/` → 页面组件
- `components/` → 可复用组件
- `data/` → 保持不变（用于本地开发）
- `constants/` → 移动到 `lib/`

### 5.4 安装依赖

```bash
npm install next@latest react@latest react-dom@latest
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### 5.5 配置 Tailwind CSS

配置 `tailwind.config.ts` 支持中文：
```typescript
module.exports = {
  content: [
    './app/**/*.{js,ts,jsx,tsx,mdx}',
    './components/**/*.{js,ts,jsx,tsx,mdx}',
  ],
  theme: {
    extend: {
      fontFamily: {
        serif: ['Noto Serif SC', 'serif'],
      },
    },
  },
  plugins: [],
}
```
