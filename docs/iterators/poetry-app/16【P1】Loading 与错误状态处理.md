# 16【P1】Loading 与错误状态处理

## 任务目标

统一处理页面 Loading 和错误状态，提升用户体验。

## 详细说明

### 16.1 Loading 状态

- 骨架屏（Skeleton）用于内容加载
- 诗词卡片骨架屏
- 详情页骨架屏

### 16.2 错误状态

- 网络错误提示
- 重试按钮
- 空状态显示

### 16.3 统一组件

创建通用组件：
- `SkeletonCard` - 卡片骨架屏
- `ErrorState` - 错误状态
- `EmptyState` - 空状态

## 验收标准

- [ ] Loading 骨架屏正常
- [ ] 错误提示正常
- [ ] 空状态正常
