# 07【P0】诗词详情页开发（中阿对照）

## 任务目标

开发诗词详情页，展示中阿对照内容，包含诗人信息、诗词正文、标签等。

## 详细说明

### 7.1 页面布局

```
┌──────────────────────────────────────────────┐
│ [< 返回]                                    │
├──────────────────────────────────────────────┤
│ 静夜思                                      │
│ 李白 · 唐                                    │
├──────────────────────────────────────────────┤
│ 【中文】                    【العربية】       │
│ ┌─────────────────────┐ ┌─────────────────┐   │
│ │ 床前明月光          │ │ ضوء القمر ...  │   │
│ │ 疑是地上霜          │ │ أعتقد أنه ...  │   │
│ │ 举头望明月          │ │ أرفع رأسي ...  │   │
│ │ 低头思故乡          │ │ أخفض رأسي ...  │   │
│ └─────────────────────┘ └─────────────────┘   │
├──────────────────────────────────────────────┤
│ 诗人简介                                    │
│ 李白（701-762），唐代诗人...                │
│诗人简介阿拉伯语...                           │
├──────────────────────────────────────────────┤
│ 标签: [思乡] [月夜] [唐诗三百首]            │
└──────────────────────────────────────────────┘
```

### 7.2 左右对照模式

- 左侧：中文原文
- 右侧：阿拉伯语译文
- 支持滑动切换单语言/双语模式

### 7.3 组件结构

- `PoemContent` - 中阿对照内容组件
- `PoetCard` - 诗人信息卡片
- `PoemTags` - 标签组件

- [ ] 标签正确显示
