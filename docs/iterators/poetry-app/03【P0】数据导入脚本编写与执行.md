# 03【P0】数据导入脚本编写与执行

## 任务目标

编写数据导入脚本，将现有的 101 首诗词从 JSON 文件导入到 Supabase 数据库。

## 详细说明

### 3.1 分析现有数据

现有数据 `data/all-translated-100-poems.json` 结构：
```json
{
  "title": "静夜思",
  "author": "李白",
  "dynasty": "唐",
  "content": ["床前明月光", "疑是地上霜", "举头望明月", "低头思故乡"],
  "content_ar": ["...", "...", "...", "..."],
  "cultural_context_ar": "...",
  "is_translated": true,
  "tags": ["思乡", "月夜"],
  "word_count": 20
}
```

### 3.2 数据映射规则

1. **朝代**：统一名称（唐、宋、元、明、清等），去"代"字
2. **诗人**：按 author 字段聚合，同一诗人多条记录只创建一条
3. **诗词类型**：全部为"诗"(shi)
4. **标签**：从 tags 字段提取，不存在则创建

### 3.3 导入脚本

创建 `scripts/import-poems.ts`：

```typescript
import { createClient } from '@supabase/supabase-js'
import poemsData from '../data/all-translated-100-poems.json'

const supabase = createClient(
  process.env.NEXT_PUBLIC_SUPABASE_URL!,
  process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!
)

async function importPoems() {
  // 1. 导入朝代（去重）
  const dynasties = [...new Set(poemsData.map(p => p.dynasty.replace('代', '')))]

  // 2. 导入诗人（去重，按朝代分组）
  const poetsMap = new Map()
  poemsData.forEach(p => {
    if (!poetsMap.has(p.author)) {
      poetsMap.set(p.author, p.dynasty.replace('代', ''))
    }
  })

  // 3. 导入诗词
  // ...

  console.log('Import completed!')
}

importPoems()
```

### 3.4 执行导入

```bash
npx ts-node scripts/import-poems.ts
```

### 3.5 验证数据

导入后执行验证 SQL：
```sql
SELECT 'dynasties' as table_name, COUNT(*) as count FROM dynasties
UNION ALL
SELECT 'poets', COUNT(*) FROM poets
UNION ALL
SELECT 'poems', COUNT(*) FROM poems
UNION ALL
SELECT 'tags', COUNT(*) FROM tags;
```

预期结果：
- dynasties: ~6 条
- poets: ~40 条
- poems: 101 条
- tags: ~20 条
