# 19【P2】深色模式支持

## 任务目标

为 poetry-app 添加深色模式支持。

## 详细说明

### 19.1 主题配置

使用 Tailwind CSS 的 dark mode：
```js
module.exports = {
  darkMode: 'class', // 或 'media'
  // ...
}
```

### 19.2 颜色方案

定义深色模式配色：
- 深色背景：`#1a1a2e`
- 深色卡片：`#16213e`
- 深色文字：`#e4e4e7`

### 19.3 主题切换

添加主题切换按钮，支持：
- 浅色模式
- 深色模式
- 跟随系统

## 验收标准

- [ ] 深色模式正常
- [ ] 主题切换正常
- [ ] 跟随系统正常
