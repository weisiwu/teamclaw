# 15【P1】阿拉伯语字体配置

## 任务目标

配置阿拉伯语字体，确保阿拉伯语内容正确显示。

## 详细说明

### 15.1 字体选择

使用 Google Fonts 的 Noto Naskh Arabic：
```html
<link href="https://fonts.googleapis.com/css2?family=Noto+Naskh+Arabic:wght@400;600;700&display=swap" rel="stylesheet">
```

### 15.2 CSS 配置

```css
.arabic-text {
  font-family: 'Noto Naskh Arabic', serif;
  direction: RTL;
  text-align: right;
}
```

### 15.3 阿拉伯语内容渲染

确保阿拉伯语内容使用正确的字体和 RTL 排版。

## 验收标准

- [ ] 阿拉伯语字体加载正常
- [ ] 阿拉伯语内容显示正确
- [ ] RTL 排版正确
