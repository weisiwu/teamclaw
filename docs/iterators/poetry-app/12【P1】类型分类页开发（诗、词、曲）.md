# 12【P1】类型分类页开发（诗、词、曲）

## 任务目标

开发诗词类型分类页面，支持按类型筛选（诗/词/曲）。

## 详细说明

### 12.1 页面布局

```
┌──────────────────────────────────────────────┐
│ Header: 分类                                  │
├──────────────────────────────────────────────┤
│ 类型筛选: [全部] [诗] [词] [曲]           │
├──────────────────────────────────────────────┤
│ 诗词列表（按筛选结果）                      │
│ ...                                          │
└──────────────────────────────────────────────┘
```

### 12.2 类型说明

- **诗**：古体诗、近体诗
- **词**：词牌名格式
- **曲**：元曲

## 验收标准

- [ ] 类型筛选正常
- [ ] 诗词列表正确
