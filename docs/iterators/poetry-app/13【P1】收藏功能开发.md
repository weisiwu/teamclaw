# 13【P1】收藏功能开发

## 任务目标

开发诗词收藏功能，支持收藏/取消收藏、收藏列表查看。

## 详细说明

### 13.1 收藏 API

创建收藏相关 API：
- `POST /api/favorites` - 添加收藏
- `DELETE /api/favorites/[poemId]` - 取消收藏
- `GET /api/favorites` - 获取收藏列表

### 13.2 用户认证

使用 Supabase Auth 实现匿名收藏（基于 localStorage）。

### 13.3 收藏状态

在诗词卡片和详情页添加收藏按钮：
```
[♡ 收藏] → [♥ 已收藏]
```

## 验收标准

- [ ] 收藏/取消收藏正常
- [ ] 收藏列表正常
- [ ] 收藏状态正确显示
