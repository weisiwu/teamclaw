# 02【P0】数据库表结构设计（细粒度）

## 任务目标

设计并创建 poetry-app 的细粒度数据库表结构，支持朝代、诗人、诗词（诗/词/曲）分类管理。

## 详细说明

### 2.1 表结构设计

#### 2.1.1 朝代表 (dynasties)

| 字段 | 类型 | 说明 |
|---|---|---|
| id | SERIAL PRIMARY KEY | 自增 ID |
| name | VARCHAR(50) UNIQUE NOT NULL | 朝代名称（如"唐"、"宋"） |
| name_arabic | VARCHAR(100) | 阿拉伯语名称 |
| period | VARCHAR(50) | 时期（如"618-907"） |
| description | TEXT | 朝代描述 |
| created_at | TIMESTAMPTZ DEFAULT NOW() | 创建时间 |

#### 2.1.2 诗人表 (poets)

| 字段 | 类型 | 说明 |
|---|---|---|
| id | SERIAL PRIMARY KEY | 自增 ID |
| name | VARCHAR(100) NOT NULL | 诗人姓名 |
| name_arabic | VARCHAR(100) | 阿拉伯语姓名 |
| dynasty_id | INTEGER REFERENCES dynasties(id) | 朝代 ID |
| bio | TEXT | 诗人简介 |
| bio_arabic | TEXT | 阿拉伯语简介 |
| created_at | TIMESTAMPTZ DEFAULT NOW() | 创建时间 |

#### 2.1.3 诗类型表 (poetry_types)

| 字段 | 类型 | 说明 |
|---|---|---|
| id | SERIAL PRIMARY KEY | 自增 ID |
| name | VARCHAR(50) UNIQUE NOT NULL | 类型名称（诗/词/曲） |
| name_arabic | VARCHAR(100) | 阿拉伯语名称 |
| description | TEXT | 类型描述 |

预设数据：
- 诗 (shi) - 古体诗、近体诗
- 词 (ci) - 词牌名格式
- 曲 (qu) - 元曲

#### 2.1.4 诗词表 (poems)

| 字段 | 类型 | 说明 |
|---|---|---|
| id | SERIAL PRIMARY KEY | 自增 ID |
| title | VARCHAR(200) NOT NULL | 标题 |
| title_arabic | VARCHAR(200) | 阿拉伯语标题 |
| poet_id | INTEGER REFERENCES poets(id) | 诗人 ID |
| poetry_type_id | INTEGER REFERENCES poetry_types(id) | 诗词类型 ID |
| content | TEXT[] NOT NULL | 正文（数组，每句一行） |
| content_arabic | TEXT[] | 阿拉伯语译文（数组） |
| translation_note | TEXT | 翻译说明 |
| cultural_context | TEXT | 文化背景 |
| cultural_context_arabic | TEXT | 阿拉伯语文化背景 |
| is_translated | BOOLEAN DEFAULT FALSE | 是否已翻译 |
| word_count | INTEGER | 字数 |
| created_at | TIMESTAMPTZ DEFAULT NOW() | 创建时间 |

#### 2.1.5 标签表 (tags)

| 字段 | 类型 | 说明 |
|---|---|---|
| id | SERIAL PRIMARY KEY | 自增 ID |
| name | VARCHAR(50) UNIQUE NOT NULL | 标签名 |
| created_at | TIMESTAMPTZ DEFAULT NOW() | 创建时间 |

#### 2.1.6 诗词-标签关联表 (poem_tags)

| 字段 | 类型 | 说明 |
|---|---|---|
| poem_id | INTEGER REFERENCES poems(id) ON DELETE CASCADE | 诗词 ID |
| tag_id | INTEGER REFERENCES tags(id) ON DELETE CASCADE | 标签 ID |
| PRIMARY KEY (poem_id, tag_id) | | 联合主键 |

### 2.2 SQL 迁移文件

创建 `supabase/migrations/001_initial_schema.sql`，包含完整的表创建语句和索引。

### 2.3 创建索引

为常用查询字段创建索引：
```sql
CREATE INDEX idx_poems_poet_id ON poems(poet_id);
CREATE INDEX idx_poems_poetry_type_id ON poems(poetry_type_id);
CREATE INDEX idx_poets_dynasty_id ON poets(dynasty_id);
```
