# 06【P0】诗词列表页开发

## 任务目标

开发诗词列表首页，支持分页、筛选、搜索功能。

## 详细说明

### 6.1 页面布局

```
┌──────────────────────────────────────────────┐
│ Header: 诗词宝库          [搜索图标]          │
├──────────────────────────────────────────────┤
│ 筛选栏: [全部] [唐] [宋] [元] [明] [清]    │
│        [诗] [词] [曲]                        │
├──────────────────────────────────────────────┤
│ 诗词卡片列表                                  │
│ ┌────────────────────────────────────────┐   │
│ │ 静夜思                                  │   │
│ │ 李白 · 唐                              │   │
│ │ 床前明月光，疑是地上霜...              │   │
│ │ [标签1] [标签2]                        │   │
│ └────────────────────────────────────────┘   │
│ ┌────────────────────────────────────────┐   │
│ │ 春晓                                    │   │
│ │ 孟浩然 · 唐                            │   │
│ │ 春眠不觉晓，处处闻啼鸟...              │   │
│ └────────────────────────────────────────┘   │
├──────────────────────────────────────────────┤
│ 分页: [< 1 2 3 ... 6 >]                   │
└──────────────────────────────────────────────┘
```

### 6.2 组件结构

- `PoemCard` - 诗词卡片组件
- `PoemFilter` - 筛选栏组件
- `Pagination` - 分页组件

### 6.3 数据获取

使用 React Query：
```typescript
const { data, isLoading } = useQuery({
  queryKey: ['poems', filters],
  queryFn: () => fetchPoems(filters),
})
```

### 6.4 筛选逻辑

- 朝代筛选：全部 / 唐 / 宋 / 元 / 明 / 清
- 类型筛选：全部 / 诗 / 词 / 曲
- 支持多选（可选）
