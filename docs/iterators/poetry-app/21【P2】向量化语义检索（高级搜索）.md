# 21【P2】向量化语义检索（高级搜索）

## 任务目标

为诗词添加向量嵌入，支持语义搜索（如"关于月亮的诗"）。

## 详细说明

### 21.1 向量化流程

1. 使用 OpenAI text-embedding-ada-002 生成诗词向量
2. 存储到 Supabase pgvector
3. 实现语义相似度搜索

### 21.2 数据库扩展

```sql
ALTER TABLE poems ADD COLUMN embedding vector(1536);
```

### 21.3 搜索 API

```typescript
// /api/poems/semantic-search
const searchEmbedding = await generateEmbedding(query)
const { data } = await supabase.rpc('match_poems', {
  query_embedding: searchEmbedding,
  match_threshold: 0.8,
  match_count: 10
})
```
