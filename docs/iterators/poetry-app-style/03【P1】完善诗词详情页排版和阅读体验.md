# 03【P1】完善诗词详情页排版和阅读体验

## 基本信息

- **任务编号**: 03
- **优先级**: P1
- **预估工时**: 45 分钟

## 详细说明

### 目标
优化诗词详情页的排版，提供更好的阅读体验。

### 具体任务

1. **标题样式**
   - 诗词标题：`text-3xl font-bold mb-4`
   - 作者信息：`text-lg text-gray-600 dark:text-gray-400 mb-8`

2. **诗词内容排版**
   - 诗句间距：`leading-relaxed` 或 `leading-loose`
   - 每句独立展示，便于阅读
   - 诗句间距：`my-4`

3. **中文翻译/注释**
   - 使用合适的中文衬线字体
   - 适当的大小和颜色

4. **页面布局**
   - 最大宽度限制：`max-w-2xl mx-auto`
   - 适当的内边距：`px-4 py-8`

5. **验证**
   - 运行 `npm run build` 确保无报错
   - 检查各设备上的阅读体验

## 验收标准

- [ ] 诗词内容排版清晰易读
- [ ] 标题和作者信息样式美观
- [ ] 页面布局合理
- [ ] `npm run build` 通过
