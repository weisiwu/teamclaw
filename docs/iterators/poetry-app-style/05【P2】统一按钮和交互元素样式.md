# 05【P2】统一按钮和交互元素样式

## 基本信息

- **任务编号**: 05
- **优先级**: P2
- **预估工时**: 30 分钟

## 详细说明

### 目标
统一所有按钮和交互元素的样式，确保一致的用户体验。

### 具体任务

1. **按钮样式标准化**
   - 主要按钮：`bg-primary text-white px-4 py-2 rounded-lg`
   - 次要按钮：`bg-gray-100 dark:bg-gray-800 text-gray-900 dark:text-gray-100 px-4 py-2 rounded-lg`
   - 危险按钮：`bg-red-500 text-white px-4 py-2 rounded-lg`
   - 幽灵按钮：`border border-gray-300 dark:border-gray-600 px-4 py-2 rounded-lg`

2. **交互元素统一**
   - 输入框：`border border-gray-300 dark:border-gray-600 rounded-lg px-4 py-2 focus:ring-2`
   - 复选框/单选框：使用统一的样式
   - 开关：使用统一样式

3. **悬停和焦点状态**
   - 按钮悬停：`hover:opacity-90 transition-opacity`
   - 输入框焦点：`focus:ring-primary focus:border-transparent`
   - 链接悬停：`hover:underline`

4. **验证**
   - 运行 `npm run build` 确保无报错
   - 检查所有交互元素样式

## 验收标准

- [ ] 按钮样式统一
- [ ] 交互元素样式一致
- [ ] 状态（悬停、焦点）正确
- [ ] `npm run build` 通过
