# 04【P2】优化阿拉伯语文本显示效果

## 基本信息

- **任务编号**: 04
- **优先级**: P2
- **预估工时**: 30 分钟

## 详细说明

### 目标
确保阿拉伯语文本在所有页面正确显示，包括 RTL 排版和字体渲染。

### 具体任务

1. **RTL 布局支持**
   - 确保阿拉伯语容器使用 `dir="rtl"`
   - 文本对齐：`text-right`
   - 列表方向：`flex-row-reverse`

2. **阿拉伯语字体**
   - 使用 `Noto Naskh Arabic` 字体
   - 备用字体：`serif, "Traditional Arabic"`
   - 字体大小：与中文保持一致或略大

3. **中阿对照排版**
   - 上下排列：中文在上，阿文在下
   - 或左右排列：中左阿右（需要容器支持）

4. **验证**
   - 运行 `npm run build` 确保无报错
   - 手动测试阿拉伯语文本显示

## 验收标准

- [ ] 阿拉伯语文本方向正确（RTL）
- [ ] 字体显示清晰
- [ ] 中阿对照排版美观
- [ ] `npm run build` 通过
