# 01【P1】统一所有页面的间距和内边距系统

## 基本信息

- **任务编号**: 01
- **优先级**: P1
- **预估工时**: 30 分钟

## 详细说明

### 目标
建立统一的间距和内边距系统，确保所有页面视觉一致性。

### 具体任务

1. **定义间距变量**
   - 在 `tailwind.config.js` 中添加统一的 spacing 扩展
   - 页面级内边距：`px-4 md:px-6 lg:px-8`
   - 卡片内边距：`p-4`
   - 元素间距：`space-y-4` / `gap-4`

2. **应用统一间距**
   - 检查所有页面：`page.tsx`、`poem/page.tsx`、`poets/page.tsx`、`dynasty/page.tsx`
   - 确保容器使用统一内边距
   - 确保卡片使用统一间距

3. **验证**
   - 运行 `npm run build` 确保无报错
   - 检查各页面间距是否一致

## 验收标准

- [ ] Tailwind 配置中添加统一的 spacing 扩展
- [ ] 所有页面容器使用统一内边距
- [ ] 所有卡片使用统一间距
- [ ] `npm run build` 通过
