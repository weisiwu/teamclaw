# 02【P1】优化诗词列表卡片样式和动画

## 基本信息

- **任务编号**: 02
- **优先级**: P1
- **预估工时**: 45 分钟

## 详细说明

### 目标
优化诗词列表页面的卡片样式，添加流畅的动画效果，提升用户体验。

### 具体任务

1. **卡片样式优化**
   - 优化卡片阴影：`shadow-sm hover:shadow-md transition-shadow`
   - 优化卡片圆角：`rounded-xl`
   - 优化卡片边框：`border border-gray-100 dark:border-gray-800`
   - 优化背景色：`bg-white dark:bg-gray-900`

2. **添加动画效果**
   - 列表项入场动画：`animate-fadeIn`
   - 悬停效果：`hover:scale-[1.02] transition-transform`
   - 点击反馈：`active:scale-[0.98]`

3. **优化布局**
   - 移动端：单列布局，`gap-4`
   - 平板端：双列布局，`grid-cols-2`
   - 桌面端：三列布局，`grid-cols-3 lg:grid-cols-4`

4. **验证**
   - 运行 `npm run build` 确保无报错
   - 检查动画流畅性

## 验收标准

- [ ] 卡片样式美观统一
- [ ] 添加流畅的动画效果
- [ ] 响应式布局正常
- [ ] `npm run build` 通过
