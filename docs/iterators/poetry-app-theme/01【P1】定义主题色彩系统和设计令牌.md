# 01【P1】定义主题色彩系统和设计令牌

## 基本信息
- **任务编号**: 01
- **优先级**: P1
- **预估工时**: 30 分钟

## 详细说明

### 目标
建立完整的主题色彩系统和设计令牌，支持古典水墨风格。

### 具体任务
1. 在 `tailwind.config.js` 中定义主题色彩变量
2. 语义化颜色命名（primary/secondary/accent）
3. 古典配色方案（墨色、宣纸白、印章红、竹青）
4. 深色模式配色

### 验收标准
- [ ] tailwind.config.js 配置完整
- [ ] 语义化颜色变量定义
- [ ] `npm run build` 通过
