# 【P1】配色方案与视觉风格全面刷新

## 背景
当前项目配色单调（灰蓝白为主），视觉风格像传统后台管理系统，缺乏个性。用户希望风格可以是现代化、可爱、酷炫中的任一种，不要当前这样的单调感。

## 目标
- 重新设计配色方案，打造有辨识度的视觉风格
- 支持多套主题切换（如：默认蓝、活力橙、暗黑紫、清新绿）
- 增加品牌感：Logo、Favicon、启动屏
- 卡片和区块增加微渐变和投影层次感
- 整体视觉从"后台管理"升级到"现代工具"

## 技术方案
1. 重新设计 CSS 变量色板（`globals.css` 中的 `:root` 变量）
2. 引入渐变色、玻璃拟态（glassmorphism）、柔和投影等现代设计元素
3. 侧边栏增加渐变背景或半透明效果
4. 卡片组件增加 hover 微动效和层次阴影
5. 页面背景从纯色改为微妙渐变或纹理
6. 支持主题切换（通过 CSS 变量 + next-themes）
7. 设计并更换 Favicon 和 Logo

## 实现文件
- `app/globals.css` — 重写色板、新增渐变/投影变量
- `tailwind.config.ts` — 扩展主题色、增加自定义阴影/渐变
- `components/layout/Sidebar.tsx` — 视觉升级
- `components/layout/Header.tsx` — 视觉升级
- `components/layout/AppLayout.tsx` — 背景升级
- `public/favicon.ico` / `public/logo.svg` — 新品牌资源

## 依赖关系
- 前置：引入 shadcn/ui 组件库（先有统一组件再刷颜色）
- 后续：动效升级

## 验证方式
1. 首次打开页面有明显的"现代感"（对比前后截图）
2. 至少支持 2 套主题色切换
3. 暗色模式配色协调、不刺眼
4. 卡片和按钮有层次感（投影/渐变）
5. **至少 5 轮修改验证**：色板设计→侧边栏/头部→卡片组件→暗色模式→主题切换
