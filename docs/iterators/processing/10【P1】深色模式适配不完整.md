# 10【P1】深色模式适配不完整

## 问题描述

部分页面硬编码了浅色模式颜色（`bg-white`、`text-gray-900`、`bg-gray-50` 等），缺少对应的 `dark:` 变体类，导致深色模式下出现白色背景块、文字不可见等问题。

## 受影响页面

### 1. `app/import/page.tsx`（最严重）

整个页面无任何 `dark:` 适配：
- 容器：`min-h-screen bg-gray-50` → 缺少 `dark:bg-slate-900`
- 所有卡片：`bg-white` → 缺少 `dark:bg-slate-800`
- 标题文字：`text-gray-900` → 缺少 `dark:text-white`
- 边框：`border-gray-200` → 缺少 `dark:border-slate-700`
- 输入框：硬编码 `border-gray-300` 未适配
- 按钮：`bg-blue-600` / `bg-gray-100` 未用设计令牌
- 步骤指示器：`bg-gray-200` / `text-gray-500` 未适配

### 2. `app/cron/page.tsx`

- `CronModal` 组件：`bg-white rounded-lg` → 缺少 `dark:bg-slate-800`
- `CronRunLogModal` 组件：`bg-white rounded-lg` → 缺少 `dark:bg-slate-800`
- 标题 `text-gray-900` → 缺少 `dark:text-white`
- label `text-gray-700` → 缺少 `dark:text-gray-300`
- textarea 边框 `border-gray-300` → 缺少 `dark:border-slate-600`
- CronCard `text-gray-900` → 缺少 `dark:text-white`

### 3. `app/agent-team/page.tsx`

- 标题 `text-gray-900` → 缺少 `dark:text-white`
- 描述 `text-gray-500` → 可保留（深色下可读）
- 派遣规则卡片 `bg-white rounded-xl border` → 缺少 `dark:bg-slate-800 dark:border-slate-700`
- 内部文字 `text-gray-700` → 缺少 `dark:text-gray-300`

### 4. `app/tokens/page.tsx`

- 标题 `text-gray-900` → 缺少 `dark:text-white`
- 描述 `text-gray-500` → 缺少 `dark:text-gray-400`

### 5. `app/monitor/page.tsx`

- 已有部分 `dark:` 适配但不完整
- 容器 `bg-gray-50` 需要 `dark:bg-slate-900`
- 卡片 `bg-white rounded-lg shadow` 需要 `dark:bg-slate-800`

## 样式方案

**样式类型：Tailwind CSS dark: 变体 + 语义化设计令牌**

### 优先使用语义化设计令牌

项目已在 `tailwind.config.ts` 和 `globals.css` 中定义了 CSS 变量驱动的语义化颜色：
- `text-foreground` 替代 `text-gray-900 dark:text-white`
- `bg-card` 替代 `bg-white dark:bg-slate-800`
- `border-border` 替代 `border-gray-200 dark:border-slate-700`
- `text-muted-foreground` 替代 `text-gray-500 dark:text-gray-400`

### 实施步骤

1. 将所有 `bg-white` → `bg-card`
2. 将所有 `text-gray-900` → `text-foreground`
3. 将所有 `border-gray-200` → `border-border`
4. 将所有 `text-gray-500` → `text-muted-foreground`
5. 对于无法用设计令牌替换的颜色，添加对应 `dark:` 变体
6. 输入框使用 `<Input>` 组件（已内建深色适配）

## 修改范围

- `app/import/page.tsx` — **全面改造**，所有硬编码颜色替换为语义令牌
- `app/cron/page.tsx` — CronModal、CronRunLogModal、CronCard 深色适配
- `app/agent-team/page.tsx` — 标题和卡片深色适配
- `app/tokens/page.tsx` — 标题区域深色适配
- `app/monitor/page.tsx` — 补充缺失的深色适配
