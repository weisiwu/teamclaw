# 18【P2】数据表格样式未使用设计系统工具类

## 问题描述

项目在 `globals.css` 中已定义了完整的表格工具类（`page-table-wrapper`、`page-table-header`、`page-table-row`、`page-table-cell` 等），但各页面的表格实现各自为政，未复用这些工具类。

## 已有表格工具类

```css
/* globals.css @layer utilities */
.page-table-wrapper { @apply bg-card border border-border rounded-xl overflow-hidden; }
.page-table-header { @apply bg-muted/50 border-b border-border px-4 py-3; }
.page-table-header-cell { @apply text-sm font-medium text-muted-foreground; }
.page-table-row { @apply hover:bg-muted/30 transition-colors; }
.page-table-cell { @apply px-4 py-3 text-sm text-foreground; }
```

## 当前表格实现对比

### 1. `app/members/page.tsx`

```tsx
<div className="bg-white dark:bg-slate-800 rounded-xl border dark:border-slate-700 overflow-hidden">
  <table className="w-full">
    <thead className="bg-gray-50 dark:bg-slate-800/50 border-b dark:border-slate-700">
      <tr>
        <th className="px-4 py-3 text-sm font-medium text-gray-600 dark:text-gray-300">...</th>
      </tr>
```

**问题**：硬编码颜色，未使用 `page-table-*` 工具类。

### 2. `app/versions/page.tsx`（紧凑视图）


**问题**：缺少外层 wrapper，部分用了语义令牌但不完整。

## 样式方案

**样式类型：Tailwind CSS 工具类替换（使用已有设计系统）**

### 统一写法

```tsx
<div className="page-table-wrapper">
  <table className="w-full">
    <thead>
      <tr className="page-table-header">
        <th className="page-table-header-cell">姓名</th>
        <th className="page-table-header-cell">角色</th>
      </tr>
    </thead>
    <tbody className="divide-y divide-border">
      {data.map(item => (
        <tr key={item.id} className="page-table-row">
          <td className="page-table-cell">{item.name}</td>
          <td className="page-table-cell">...</td>
        </tr>
      ))}
    </tbody>
  </table>
</div>
```

### 斑马纹（可选）

如需保留 `members/page.tsx` 的斑马纹效果，可在 `globals.css` 添加：

```css
.page-table-row-striped:nth-child(even) {
  @apply bg-muted/20;
}
```

## 修改步骤

1. `app/members/page.tsx`：表格外层 → `page-table-wrapper`，thead → `page-table-header`，th → `page-table-header-cell`，tr → `page-table-row`，td → `page-table-cell`
2. `app/versions/page.tsx`（紧凑视图）：添加 `page-table-wrapper`，统一 th/td 类名
3. 验证深色模式和响应式表现

## 修改范围

- `app/members/page.tsx` — 表格样式迁移到工具类
- `app/versions/page.tsx` — 紧凑视图表格样式迁移
