# 08【P1】登录弹框样式异常 — 未全屏遮罩

## 问题描述

当前登录页的登录弹框样式异常，弹框没有覆盖整个页面（缺少全屏遮罩层），导致用户可以看到背后的页面内容，视觉体验不佳。

## 预期行为

- 登录弹框应有全屏半透明遮罩（overlay），覆盖整个视口
- 弹框居中显示在遮罩之上
- 遮罩阻止用户与背后页面交互

## 当前行为

- 登录弹框未覆盖整个页面
- 背后页面内容可见且可能可交互

## 涉及文件

> 待定位，可能涉及：
> - 登录页组件（`app/login/` 或相关页面）
> - 登录弹框组件
> - 全局 layout 中的认证拦截逻辑

## 修复方向

1. 定位登录弹框组件
2. 添加 `fixed inset-0` 全屏遮罩层（如使用 Tailwind）
3. 确保 `z-index` 足够高，覆盖所有页面内容
4. 遮罩背景使用半透明黑色（如 `bg-black/50`）

## 优先级

P1 — 影响用户首次使用体验
