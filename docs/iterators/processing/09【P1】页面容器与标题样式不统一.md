# 09【P1】页面容器与标题样式不统一

## 问题描述

各页面的容器包裹方式和标题样式不一致，存在多种写法，视觉上出现间距和字号不统一的问题。

### 当前状况

| 页面 | 容器写法 | 标题写法 |
|------|----------|----------|
| `app/page.tsx` | `page-container` | `text-3xl font-bold` |
| `app/versions/page.tsx` | `page-container` | `page-header-title` |
| `app/capabilities/page.tsx` | `page-container` | `text-3xl font-bold` |
| `app/members/page.tsx` | `page-container` | `text-2xl font-bold` |
| `app/agent-team/page.tsx` | `p-6 space-y-6` | `text-2xl font-bold` |
| `app/tokens/page.tsx` | `p-6 space-y-6` | `text-2xl font-bold` |
| `app/cron/page.tsx` | `p-6 space-y-6` | `text-2xl font-bold` |
| `app/docs/page.tsx` | `p-6 max-w-7xl mx-auto` | `text-2xl font-bold` |
| `app/monitor/page.tsx` | `min-h-screen bg-gray-50 p-8` | `text-3xl font-bold` |
| `app/import/page.tsx` | `min-h-screen bg-gray-50 py-8` | `text-2xl font-bold` |
| `app/settings/page.tsx` | 无容器 class | `text-3xl font-bold` |

## 期望效果

- 所有页面统一使用 `page-container` 工具类（已在 `globals.css` 中定义为 `@apply p-4 sm:p-6 space-y-6`）
- 所有页面标题统一使用 `page-header` + `page-header-title` 工具类
- 副标题统一使用 `page-header-subtitle`

## 样式方案

**样式类型：Tailwind CSS 工具类（已有设计系统）**

已在 `app/globals.css` `@layer utilities` 中定义：

```css
.page-container { @apply p-4 sm:p-6 space-y-6; }
.page-header { @apply flex items-center justify-between mb-6; }
.page-header-title { @apply text-2xl font-bold text-foreground; }
.page-header-subtitle { @apply text-sm text-muted-foreground mt-1; }
```
