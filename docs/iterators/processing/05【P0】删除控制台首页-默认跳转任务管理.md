# 40【P0】删除控制台首页 - 默认跳转任务管理

## 背景
当前控制台（`/`）仅展示统计卡片和菜单跳转链接，无独立业务价值，增加了一层无意义的跳转。用户反馈"控制台没有业务意义"。

## 目标
- 删除控制台首页
- 访问 `/` 时直接跳转到任务管理 `/tasks`
- 从侧边栏移除「控制台」入口

## 技术方案
1. 修改 `app/page.tsx` 为 redirect 到 `/tasks`
2. 从 `Sidebar.tsx` 移除控制台 menuItem
3. 删除控制台相关的 dashboard API 调用（如果其他页面不依赖）

## 实现文件
- `app/page.tsx` — 改为 redirect
- `components/layout/Sidebar.tsx` — 移除控制台入口
