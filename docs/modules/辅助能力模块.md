# 辅助能力模块

[← 返回主文档](../系统架构.V1.md)

---

## 1. 概述

面向所有角色用户的自助查询功能，通过与 pm 对话或后台界面使用。

---

## 2. 能力清单

| 能力 | 说明 | 使用方式 |
|------|------|---------|
| 查看项目文档库 | 群聊文档自动上传归档，可浏览 | @pm "查看文档" 或 后台界面 |
| 查看项目详情 | 技术栈、架构、状态总览 | @pm "项目详情" 或 后台界面 |
| 查看任务列表 | 所有任务及其状态 | @pm "任务列表" 或 后台界面 |
| 查看具体任务 | 任务详情、变更记录、commit | @pm "查看任务 t_001" 或 后台界面 |
| 查看产物列表 | 所有构建版本和产物 | @pm "产物列表" 或 后台界面 |
| 下载文件 | 文档和构建产物下载 | 后台界面 |
| 后台配置操作 | 仅管理员/副管理员可用 | 后台界面 |

---

## 5. 权限控制

### 5.1 普通员工权限

普通员工只能使用辅助能力，不能发布任务：

```
可用功能：
  ✅ 查看项目文档库
  ✅ 查看项目详情
  ✅ 查看任务列表
  ✅ 查看具体任务
  ✅ 查看产物列表
  ✅ 下载文件
  ❌ 后台配置操作
  ❌ 发布任务
  ❌ 完成/取消任务
```

### 5.2 管理员/副管理员权限

管理员和副管理员拥有所有权限：

```
可用功能：
  ✅ 所有辅助能力
  ✅ 后台配置操作
  ✅ 发布任务
  ✅ 完成/取消任务
  ✅ 版本管理
  ✅ 人员管理
```

## 6. 文档自动归档

群聊中的文档消息会自动上传归档：

```
用户在群里发送文件
    │
    ▼
Layer1 接收文件消息
    │
    ├── 1. 下载文件到本地
    ├── 2. 识别文件类型
    ├── 3. 转换为在线可浏览格式
    ├── 4. 存储到文档库
    ├── 5. 向量化（如果是文本文档）
    │
    ▼
pm 在群内回复
    pm: "文件已收到并归档：产品需求文档.docx
         查看：http://localhost:9700/docs/产品需求文档.docx"
```

## 7. 搜索功能

### 7.1 文档搜索

支持全文搜索和语义搜索：

```
用户: @pm 搜索文档 "收藏功能"

pm: 找到 3 个相关文档：
    1. 产品需求文档.docx - "用户可以收藏喜欢的诗词"
    2. API 接口文档.md - "POST /api/favorites 收藏接口"
    3. 技术设计.md - "收藏功能使用 Zustand 管理状态"
```

### 7.2 任务搜索

支持按标题、标签、文件名搜索：

```
用户: @pm 搜索任务 "首页"

pm: 找到 5 个相关任务：
    1. t_20260316_001 - 首页按钮样式修改 [已完成]
    2. t_20260315_002 - 首页性能优化 [已完成]
    3. t_20260314_005 - 首页布局调整 [已完成]
    ...
```
