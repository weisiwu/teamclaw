# 辅助能力模块

[← 返回主文档](../系统架构.V1.md)

---

> ⚠️ **数据源要求**：严格禁止使用 mock 数据或伪造数据。所有数据必须来自真实项目仓库（默认 https://github.com/weisiwu/teamclaw ）。数据源可在 [项目导入模块](./项目导入模块.md) 中通过「URL 下载」或「本地上传」方式切换。
>
> 🎨 **样式检测要求**：每个功能在实现过程中，必须检测并验证实际渲染的 UI 样式（布局、间距、配色、响应式等），确保页面展示效果与设计一致，禁止只关注逻辑而忽略样式。

---

## 1. 概述

面向所有角色用户的自助查询功能，通过与 pm 对话或后台界面使用。

---

## 2. 能力清单

| 能力 | 说明 | 使用方式 |
|------|------|---------|
| 查看项目文档库 | 群聊文档自动上传归档，可浏览 | @pm "查看文档" 或 后台界面 |
| 查看项目详情 | 技术栈、架构、状态总览 | @pm "项目详情" 或 后台界面 |
| 查看任务列表 | 所有任务及其状态 | @pm "任务列表" 或 后台界面 |
| 查看具体任务 | 任务详情、变更记录、commit | @pm "查看任务 t_001" 或 后台界面 |
| 查看产物列表 | 所有构建版本和产物 | @pm "产物列表" 或 后台界面 |
| 下载文件 | 文档和构建产物下载 | 后台界面 |
| 后台配置操作 | 仅管理员/副管理员可用 | 后台界面 |

## 5. 权限控制

### 5.1 普通员工权限

普通员工只能使用辅助能力，不能发布任务：

```
可用功能：
  ✅ 查看项目文档库
  ✅ 查看项目详情
  ✅ 查看任务列表
  ✅ 查看具体任务
  ✅ 查看产物列表
  ✅ 下载文件
  ❌ 后台配置操作
  ❌ 发布任务
  ❌ 完成/取消任务
```

### 5.2 管理员/副管理员权限

管理员和副管理员拥有所有权限：

```
可用功能：
  ✅ 所有辅助能力
  ✅ 后台配置操作
  ✅ 发布任务
  ✅ 完成/取消任务
  ✅ 版本管理
  ✅ 人员管理
```

## 6. 文档自动归档

群聊中的文档消息会自动上传归档：

```
用户在群里发送文件
    │
    ▼
Layer1 接收文件消息
    │
    ├── 1. 下载文件到本地
    ├── 2. 识别文件类型
    ├── 3. 转换为在线可浏览格式
    ├── 4. 存储到文档库
    ├── 5. 向量化（如果是文本文档）
    │
    ▼
pm 在群内回复
    pm: "文件已收到并归档：产品需求文档.docx
         查看：http://localhost:9700/docs/产品需求文档.docx"
```

## 7. 搜索功能

### 7.1 文档搜索

支持全文搜索和语义搜索：

```
用户: @pm 搜索文档 "收藏功能"

pm: 找到 3 个相关文档：
    1. 产品需求文档.docx - "用户可以收藏喜欢的诗词"
    2. API 接口文档.md - "POST /api/favorites 收藏接口"
    3. 技术设计.md - "收藏功能使用 Zustand 管理状态"
```

### 7.2 任务搜索

支持按标题、标签、文件名搜索：

```
用户: @pm 搜索任务 "首页"

pm: 找到 5 个相关任务：
    1. t_20260316_001 - 首页按钮样式修改 [已完成]
    2. t_20260315_002 - 首页性能优化 [已完成]
    3. t_20260314_005 - 首页布局调整 [已完成]
    ...
```

## 8. API 定义

| 方法 | 路径 | 说明 |
|------|------|------|
| `GET` | `/api/v1/abilities` | 获取辅助能力列表（含启用/禁用状态） |
| `PUT` | `/api/v1/abilities/:abilityId/toggle` | 启用/禁用指定能力 |
| `GET` | `/api/v1/docs` | 获取项目文档库列表（支持分页、搜索） |
| `GET` | `/api/v1/docs/:docId` | 获取文档内容（在线浏览） |
| `GET` | `/api/v1/docs/:docId/download` | 下载文档原始文件 |
| `GET` | `/api/v1/search/docs` | 文档全文搜索 + 语义搜索 |
| `GET` | `/api/v1/search/tasks` | 任务搜索（按标题、标签、文件名） |
| `GET` | `/api/v1/artifacts` | 获取构建产物列表 |
| `GET` | `/api/v1/artifacts/:artifactId/download` | 下载构建产物 |

**GET /api/v1/abilities 响应示例**

```json
{
  "code": 0,
  "data": {
    "list": [
      { "id": "view_docs", "name": "查看项目文档库", "enabled": true },
      { "id": "view_project", "name": "查看项目详情", "enabled": true },
      { "id": "view_tasks", "name": "查看任务列表", "enabled": true },
      { "id": "view_artifacts", "name": "查看产物列表", "enabled": true },
      { "id": "download_file", "name": "下载文件", "enabled": false },
      { "id": "admin_config", "name": "后台配置操作", "enabled": true }
    ]
  },
  "message": "ok"
}
```

**GET /api/v1/search/docs 请求示例**

```
GET /api/v1/search/docs?q=收藏功能&type=semantic&page=1&pageSize=10
```

## 9. 实现任务清单

> 前置依赖：与版本管理模块同属阶段 5，可并行开发，但需要阶段 4 验收通过。

### 任务 9.1：后端 - 辅助能力管理

| # | 任务 | 产出文件 | 说明 |
|---|------|---------|------|
| 1 | 能力配置数据模型 | `server/src/models/ability.ts` | Ability 类型定义、默认能力列表 |
| 2 | 辅助能力路由 | `server/src/routes/ability.ts` | 能力列表 + 开关切换 API |
| 3 | 能力管理服务 | `server/src/services/abilityService.ts` | 能力状态管理、权限校验（管理员才能切换） |

### 任务 9.2：后端 - 文档与搜索

| # | 任务 | 产出文件 | 说明 |
|---|------|---------|------|

## 10. 验收标准

| # | 验收项 | 验证方式 |
|---|--------|---------|
| 1 | 能力管理页面正确展示所有能力，开关切换后状态持久化 | 浏览器操作 |
| 2 | 文档库页面列出所有已归档文档，上传时间正确 | 浏览器截图 |
| 3 | 文档在线浏览正常，Markdown 渲染正确，代码语法高亮 | 浏览器截图 |
| 4 | 文档下载功能正常，文件完整 | 浏览器下载验证 |
| 5 | 全文搜索返回相关文档片段，关键词高亮 | 浏览器操作 |
| 6 | 语义搜索返回语义相关结果（非精确匹配） | API 验证 |
| 7 | 任务搜索按标题/标签/文件名正确过滤 | 浏览器操作 |
| 8 | 产物列表页展示所有版本产物，下载链接可用 | 浏览器下载验证 |
| 9 | 普通员工只能使用辅助查询，不能操作后台配置 | 权限验证 |
| 10 | 所有页面样式正确，搜索结果布局合理，空状态展示正确 | 浏览器样式检查 |
