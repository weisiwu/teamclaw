## 2. е®үиЈ…жөҒзЁӢ

### 2.1 зі»з»ҹиҰҒжұӮ

- **ж“ҚдҪңзі»з»ҹ**пјҡmacOS / Linux / Windows
- **Node.js**пјҡ>= 18.0.0
- **Python**пјҡ>= 3.10
- **Git**пјҡ>= 2.30
- **Docker**пјҡ>= 20.10пјҲеҸҜйҖүпјҢз”ЁдәҺж•°жҚ®еә“пјү

### 2.2 е®үиЈ…жӯҘйӘӨ

```bash
# 1. е®үиЈ… TeamClaw CLI
npm install -g @teamclaw/cli

# 2. еҲқе§ӢеҢ–е·ҘдҪңз©әй—ҙ
teamclaw init

# 3. е®үиЈ…дҫқиө–
cd ~/.openclaw
npm install

# 4. еҗҜеҠЁж•°жҚ®еә“пјҲDocker ж–№ејҸпјү
docker-compose up -d

# 5. еҗҜеҠЁеҗҺеҸ°жңҚеҠЎ
teamclaw serve
# вҶ’ жөҸи§ҲеҷЁжү“ејҖ http://localhost:9700

# 6. еҜје…ҘйЎ№зӣ®
# еңЁ Web еҗҺеҸ°йҖүжӢ©йЎ№зӣ®и·Ҝеҫ„пјҢејҖе§ӢеҲқе§ӢеҢ–
```

## 4. Docker Compose й…ҚзҪ®

`~/.openclaw/docker-compose.yml`

```yaml
version: '3.8'

services:
  postgres:
    image: postgres:15
    container_name: teamclaw-postgres
    environment:
      POSTGRES_DB: teamclaw
      POSTGRES_USER: teamclaw
      POSTGRES_PASSWORD: your_password
    ports:
      - "5432:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data

  redis:
    image: redis:7
    container_name: teamclaw-redis
    ports:
      - "6379:6379"
    volumes:
      - redis_data:/data

  chromadb:
    image: chromadb/chroma:latest
    container_name: teamclaw-chromadb
    ports:
      - "8000:8000"
    volumes:
      - chromadb_data:/chroma/chroma

volumes:
  postgres_data:
  redis_data:
  chromadb_data:
```

## 7. Nginx й…ҚзҪ®

`/etc/nginx/sites-available/teamclaw`

```nginx
server {
    listen 80;
    server_name teamclaw.local;

    # еүҚз«ҜйқҷжҖҒж–Үд»¶
    location / {
        root /home/user/.openclaw/dashboard/dist;
        try_files $uri $uri/ /index.html;
    }

    # API д»ЈзҗҶ
    location /api {
        proxy_pass http://localhost:9700;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }

    # WebSocket д»ЈзҗҶ
    location /ws {
        proxy_pass http://localhost:9700;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "Upgrade";
        proxy_set_header Host $host;
    }
}
```

